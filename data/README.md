# Folder `data/`

Folder ini digunakan untuk menyimpan dataset lokal (opsional, tidak diinclude di repo).

## Dataset yang digunakan

**CDD — Comprehensive Disaster Dataset**

- **Sumber:** Kaggle — [CDD Dataset by Niloy et al. (2021)](https://www.kaggle.com/datasets/nitishabharathi/scene-classification)
- **Total citra:** 13.557 gambar
- **Kelas:** 6 (Damaged_Infrastructure, Fire_Disaster, Human_Damage, Land_Disaster, Non_Damage, Water_Disaster)
